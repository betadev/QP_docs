Users

This section lets add / delete users and set access permissions for them.

Creating New User
  • Click the Configure Settings button on the app's main screen.
  • Click Configure Users.
  • Select 'Create New User' tab.
  • Enter email id and password for new user and click the 'Create User' button.
Setting User permissions
  • Click the Configure Settings button on the app's main screen.
  • Click Configure Users.
  • Select 'Set User Access' tab.
  • Select email id of the user you want to edit and click 'Edit User Permissions' button.
  • Select Yes / No for the different sections listed under 'Section Access Permissions' to grant / revoke access to a particular section.
  • Set appropriate permissions for the different operations listed under 'Operation Access Permissions' to grant permissions as required.
  • Set appropriate permissions in 'Receive Disruption Alerts' section to subscribe users for in-app & email alerts of that particular operation.
  • Click the 'Save Changes' button.
Note: 'Update' permission allows read, write and update access as well as permission to Reject parts or allow Deviations.
Deleting User
  • Click the Configure Settings button on the app's main screen.
  • Click Configure Users.
  • Select 'Set User Access' tab.
  • Select email id of the user you want to edit and click 'Edit User Permissions' button.
  • Click the 'Delete User' button and confirm.
