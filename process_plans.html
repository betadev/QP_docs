Process Plans

Each Process Plan is a set of operations with certain specifications that define a particular product model. Add operations and workstations used across all your different processes before configuring Process Plans.

Example: Suppose you make tables of 3 different sizes. You would have to create a Process Plan for each different table model-

  • 4 seater table
  • 6 seater table
  • 8 seater table

Adding / Editing Process Plans
  • Click the Configure Settings button on the app's main screen.
  • Click Setup Process Plans.
  • Enter a Model name for a new Process Plan or select a Model name from the dropdown menu if it already exists and click 'Continue'.
  • Use the '+ Operation' / '- Operation' buttons to add / remove the different Operation Steps required in the process plan.
  • For each step, select an Operation Name and enter it's cycle time (in minutes) [optional] for the particular Model.
  • Use the '+ Parameter' / '- Parameter' buttons to add / remove the parameters for each Operation Step.
  • Fill in the parameter info (Check notes below for help) across the different Operation Steps.
  • Click the 'Save Process Plan'.
Each Parameter type has the following fields
  • Parameter Name [Required] :
    • Enter name / description of the particular operation parameter.
  • Parameter Criticality [Required] :
    • Major criticality - does not allow data entry & process execution to continue until a deviation is allowed.
    • Minor criticality - allows data entry to continue for other operation steps and a deviation can be provided anytime during the process execution.
    • When some parameter value is not within the acceptable range, a 'Deviation Required' warning message is displayed in the 'Basic Info' section and the Operation Step where the parameter is present.
  • Parameter Reference Link [Optional] :
    • URL link for a document, image or anything else that provides descriptive information about the parameter. This information is displayed as a hint while entering parameter data.
    • URL link for documents / images can be obtained by uploading your files to google drive, dropbox, github or any other cloud platform from where you can obtain a public sharing url.
  • Measurement Method [Optional] :
    • Enter any device / tool / method name used to measure the parameter. Eg - scale, vernier caliper, visual, etc.
  • Parameter Types [Required] :
    • Numeric Range -
      • Specify minimum and maximum numeric values.
      • Any value beyond specified range will require a deviation.
    • Option List -
      • Specify a list of acceptable & unacceptable values.
      • Any unacceptable value entered will require a deviation.
    • Sub-Job -
      • Specify model type of a job / sub-component / part which is to be linked with current job.
        Example - linking sub components to a main assembly.
      • Any job / part serial number which belongs to a different model type or is invalid wil require a deviation.
    • Free Response -
      • No specification required.
      • Allows user to input any value (max 50 characters).
