Analytics & Reports

Qik Process provides powerful reports that let you monitor your process efficiency and quality through simple and easy to understand graphs and tables. You can analyze realtime / past data and apply filters to quickly discover trends at operation, workstation, and user levels. Check out the different reports available to learn more.

Realtime Operation Analytics

This report provides a realtime summary of operation for the current day (12:00 am to 11:59 pm). The report consists of 2 graphs-

  • Jobs Pending per Operation - This graph displays the total number of jobs pending in a particular operation
  • Jobs Completed per Operation - This graph display the total number of jobs that were completed / rejected in a particular operation
To view the Realtime Operation Analytics Report-
  • Click the Dashboard button on the app's main screen.
  • Click the Realtime Operation Analytics button.
  • Click Fetch Data.
  • Select 'Graphical View' tab or 'Table View' tab as required.

Daily Operation Analytics

This report provides detailed daily statistics of an operation for a selected period. To view the Daily Operation Analytics Report-

  • Click the Dashboard button on the app's main screen.
  • Click the Daily Operation Analytics button.
  • Select 'Start Date', 'End Date', 'Operation Name' and click the Get Data button.
  • Select 'Graphical View' tab or 'Table View' tab as required.
  • In 'Graphical View', you can filter data by date, model, status, workstation, etc by checking / unchecking items in the dropdown fields.

Hourly Operation Analytics

This report provides detailed hourly statistics of an operation for a selected period. To view the Hourly Operation Analytics Report-

  • Click the Dashboard button on the app's main screen.
  • Click the Hourly Operation Analytics button.
  • Select 'Start Date', 'End Date', 'Operation Name' and click the Get Data button.
  • Select 'Graphical View' tab or 'Table View' tab as required.
  • In 'Graphical View', you can filter data by date, model, status, workstation, etc by checking / unchecking items in the dropdown fields.

Pending Jobs by Operation

This report displays the amount of time jobs are pending in a particular operation stage and can be used to find overdue jobs. To view the Pending Jobs by Operation Report-

  • Click the Dashboard button on the app's main screen.
  • Click the Pending Jobs by operation button.
  • Select 'Operation Name' and click the Fetch Data button.

Jobs Requiring Deviations

This report displays the list of jobs that require a deviation and the time since the jobs are pending. To view the Jobs Requiring Deviations Report-

  • Click the Dashboard button on the app's main screen.
  • Click the Jobs Requiring Deviations button.
  • Click the Fetch Data button.

Download Job Records

This report is used to view / export job records between 2 serial numbers. To view the Download Job Records Report-

  • Click the Dashboard button on the app's main screen.
  • Click the Download Job Records button.
  • Enter the 'Starting Serial Number' and 'Ending Serial Number' and click the Download Job Records button.

Other Usage Notes:
  • For Graphs- You can select the dropdown present above the graphs to view data grouped by model, workstation, status, user id, etc. You can also click on any of the categories in the graph's legend to show / hide data from that category group.
  • For Tables- You can click "Select Viewing Mode" to show or hide parameters from a particular operation. You can also use the "Search Builder" to filter data as per your criteria or use the "Export Data" button to view the data using some other app.
